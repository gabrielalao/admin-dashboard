## Install Guide

This backend based on Node.js & MySQL

### Install NPM

Using following bash command you can install and run it on your local.

```bash
$ npm install
```

```bash
$ npm start
```

For development
```bash
$ npm run serve
```

Change config for database
(\config.json)
